== Description ==

Adds Meta keywords and Description to each and every page, post, archive.

Uses Title, Description, Categories and tags for keywords generation. Uses content after striping short codes and short tags for generation of description.

Reduces manual work of adding them manually.


== Installation ==


1. Upload the zipped file to yoursite/wp-content/plugins
2. Activate the plugin through the 'Plugins' menu in WordPress
